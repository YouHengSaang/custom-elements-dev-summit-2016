Custom Elements

Create custom DOM elements. <esri-web-map>.

Since every framework must work with regular DOM elements they must work with custom elements.

  id="map"
  basemap="NightVector"
  zoom="5"
  lat="38.95"
  lng="-100.95"></esri-map>
